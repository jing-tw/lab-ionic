# Create simple button

## Install
```bash
npm install
ionic build
```

## Run
```bash 
echo 'plug-in the Android device'  # if you use VBox, check the [Devices]
ionic cordova run android --host 192.168.21.3  # test on real device
ionic serve  # test on previewing
```
